# 社交网络关系分析系统 (SocialBondsAnalysis)

这是一个基于 C++ 开发的社交网络分析程序，旨在通过底层数据结构实现人员管理及社交关系模拟。本项目最核心的特点是：**完全不依赖 C++ 标准模板库 (STL)**，所有基础容器均为手动实现，深度展示了内存管理与底层逻辑。

## 🚀 项目进展

目前已完成底层基建工程：

* [x] **MyString**: 手搓字符串类，处理深拷贝与内存释放。
* [x] **MyVector**: 动态数组模板类，支持自动扩容。
* [x] **MyQueue**: 链表队列模板类，为图的 BFS 算法提供弹药。
* [x] **Person**: 社交网络中的人员实体类。
* [ ] **SocialNetwork**: 社交网络大脑（正在开发中，包含邻接矩阵与 BFS 算法）。

## 🛠️ 技术亮点：底层基建

### 1. 内存管理 (The Big Three)

在 `MyString` 和 `MyVector` 的实现中，严格遵循了 C++ 的“三法则”：

* **析构函数**：确保所有 `new` 出来的内存被安全 `delete`。
* **拷贝构造函数**：实现深拷贝，防止多个对象共享同一块指针导致崩溃。
* **赋值运算符重载**：处理自赋值检查与旧内存清理。

### 2. 自定义容器

* **MyVector**: 采用倍增扩容策略，提供 `O(1)` 的随机访问效率。
* **MyQueue**: 基于单链表实现，支持 `O(1)` 的入队与出队，且利用模板化设计屏蔽了对具体数据类型的依赖。

### 3. 防御性编程

* 在容器中实现了 `const` 成员函数的双重重载，确保只读对象的数据安全。
* 对队列和动态数组的边界条件（如空队列弹出、越界访问）进行了逻辑保护。

## 🏗️ 如何运行

本项目使用 CMake 进行构建。

1. 环境要求：支持 C++11 及以上标准的编译器 (GCC/Clang/MSVC)。
2. 编译步骤：
```bash
mkdir build && cd build
cmake ..
make
./SocialBondsAnalysis

```
